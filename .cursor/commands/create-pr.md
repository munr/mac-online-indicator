## create-pr

If I'm on the main branch and run this command, tell me I can't do run it on the main branch and it has to be run on a feature branch.

If I'm not on the main branch, push the branch and create a new PR to merge this branch to main. Generate the PR title and description by comparing the changes to main and summarizing the changes that were made.

After the PR is created, open it in my external browser.
